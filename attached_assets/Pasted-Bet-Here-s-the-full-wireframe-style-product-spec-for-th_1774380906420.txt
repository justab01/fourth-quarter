Bet. Here’s the full wireframe-style product spec for the sports board system so it stays easy, clickable, and never feels like too much.

FOURTH QUARTER

Universal Sports Navigation System

Goal: let users see all sports without getting lost, while making every league, team, player, and live event fast to reach.

⸻

1. CORE PRODUCT IDEA

This app should feel like:
	•	broad at the top
	•	specific when tapped
	•	clean the whole way down

User flow:

Home
→ Sport Group
→ League / Tour / Competition
→ Event / Team / Player
→ Details

Example:

Home
→ Golf
→ LIV
→ Miami Event
→ Leaderboard / Players / News

Or:

Home
→ Soccer
→ Champions League
→ Arsenal vs Madrid
→ Stats / Lineups / News / Reactions

That’s the whole mental model.

3. SCREEN SYSTEM

A. HOME SCREEN

This is the “don’t overwhelm me” page.

Top section
	•	Search bar
	•	profile icon
	•	notifications icon
	•	live now button

Section order

1. Live Now
Horizontal cards showing current active games/events across all sports.

Example cards:
	•	Lakers vs Suns — Q4
	•	LIV Miami — Round 2 leaderboard
	•	Arsenal vs Madrid — 67’
	•	UFC Fight Night — Round 3
	•	Astros vs Rangers — Top 8

2. Biggest Stories
Big hero cards for the most important sports moments.

3. Favorites
Only followed teams, leagues, players, or events.

4. Trending
Fast-moving buzz, highlights, arguments, upset alerts.

5. Browse Sports
Grid of sport icons/cards.

6. Upcoming
Today / tonight / this week.

12. CONTENT PRIORITY LOGIC

The app should rank content like this:

Tier 1
	•	live events
	•	breaking news
	•	finals / playoffs / majors / title fights
	•	major player injuries or transactions

Tier 2
	•	top results
	•	standings movement
	•	trend stories
	•	schedule previews

Tier 3
	•	niche stories
	•	deeper analysis
	•	smaller league updates

So users see the biggest stuff first without losing access to everything else.

16. WHAT NOT TO DO

Do not:
	•	dump every sport on one endless feed
	•	hide leagues too deep
	•	make users guess where things are
	•	over-brand common sports labels
	•	make search article-only
	•	make all cards equal importance
	•	force 5 taps to get to something obvious like LIV

Need to feel:
	•	obvious
	•	fast
	•	clean
	•	layered

18. THE MAIN PRODUCT SENTENCE

Here’s the clean line for your build team:

The Fourth Quarter should organize the entire sports world into a layered board system where users can move from all sports to specific leagues, teams, players, and live events in as few taps as possible, with strong personalization and clear visual hierarchy.
