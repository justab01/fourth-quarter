Yeah — don’t make it “more sports,” make it better organized sports.

What you need is a sports universe with layers so nobody gets drowned in chaos. The goal is:

see everything
but also
only feel what matters right now

The fix: build it like a board system

Think of the whole thing as a main sports home, then every sport opens into its own clean lane.

1. Main Home = “All Sports Board”

This is the top layer. It should show every sport without looking crowded.

At the top:
	•	Live Now
	•	Biggest Headlines
	•	Your Favorites
	•	Trending Games
	•	Watchlist

Then under that, clickable sport tiles:
	•	NBA
	•	NFL
	•	MLB
	•	NHL
	•	College Basketball
	•	College Football
	•	Soccer
	•	Golf
	•	Tennis
	•	UFC / Boxing
	•	WNBA
	•	Motorsports
	•	Track & Field
	•	Esports
	•	More

Each one should feel like a big clean button/card, not tiny text links.

So if somebody wants LIV Golf, they tap Golf first.

2. Every sport gets its own “sub-board”

This is the part you’re really talking about.

When somebody taps Golf, it opens a clean board with horizontal tabs or pills at the top:

	•	Rankings

	•	Results

4. Use “boards within boards”

This is the cleanest mental model.

For example:

Sports
→ Golf
→ LIV
→ Event
→ Leaderboard / Players / Highlights / News

Or:

Sports
→ Soccer
→ Champions League
→ Match
→ Stats / Lineups / Table / Reactions

That way users always know where they are.
No confusion.
No “where do I find this?”

⸻

5. The card system needs hierarchy

Not every item should look equally important.

You want 3 levels:

Hero cards

For the biggest stuff.
Big image, score, headline, live badge.

Standard cards

For normal games/news/results.

Mini cards

For quick scores, rankings, smaller updates.

That way the screen breathes.
Everything not screaming at the same volume.

⸻

6. “Live” needs its own mode

This is huge.

There should be a Live rail or Live tab that pulls from all sports:
	•	Lakers vs Suns
	•	LIV event leaderboard
	•	Champions League match
	•	UFC prelim result

So users don’t need to hunt.
They can see what’s active right now.

Then tap in deeper if they care.

9. Best layout pattern for mobile

Mobile should be:

Top:
	•	Search bar
	•	Live toggle
	•	Favorites toggle

Below:
	•	horizontal sport chips

Below:
	•	stacked cards

Inside each sport board:
	•	sticky sub-tabs
	•	quick filters
	•	scrollable sections

So for Golf:
	•	top chips: PGA / LIV / Majors / Players
	•	then cards: leaderboard, event, news, trending names

No deep menus every 2 seconds.
Mostly tap and scroll.
