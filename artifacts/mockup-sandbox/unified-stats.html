Design System

Spider Chart (Team Identity)
  • • 5-6 sport-specific metrics
  • • Instantly see team strengths/weaknesses
  • • Team color fill + border
Progress Bars (Detail)
  • • Exact values + league ranking
  • • Color-coded by rank tier
  • • Gradient for top stats
Sport-Specific Stats
  • • NBA: PPG, 3PT%, DEF RTG, REB, TOV
  • • NFL: PASS YDS, 3RD%, RZ%, RUSH, TO DIFF
  • • MLB: HR, RPG, AVG, ERA, SAVE%
  • • NHL: GAA, SV%, G/GP, PP%, PK%
  • • Soccer: GF, xG, POSS, GA, CS
